# Geometrische Konstruktionszeichnungen

Erste Struktur für eine lokal laufende Unterrichts-Webseite zu geometrischen Grundkonstruktionen.

## Inhalt

- `index.html`  
  Übersichtsseite mit Konstruktionskarten.

- `konstruktionen/mittelsenkrechte.html`  
  Erste ausgearbeitete Konstruktionsseite mit Schritt-für-Schritt-Animation.

- `css/style.css`  
  Gemeinsames helles Layout, Karten, Buttons, SVG-Strichkonzept und Legende.

- `js/mittelsenkrechte.js`  
  Schrittlogik für die Konstruktion der Mittelsenkrechten.

## Strichkonzept

- Gegebene Punkte: dunkle Punkte
- Gegebene Strecken: kräftige dunkle Linie
- Hilfslinien und Hilfskreise: dünn, grau, gestrichelt
- Neu konstruierte Punkte: blau
- Fertige Konstruktion: kräftig grün

Die Legende steht immer direkt unter der jeweiligen Konstruktion.

## Nutzung

1. ZIP-Datei entpacken.
2. `index.html` im Browser öffnen.
3. Auf „Mittelsenkrechte“ klicken.

Die Seite benötigt keine Internetverbindung.
